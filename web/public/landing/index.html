One repo. Every team. Any number of agents.

Runko is a self-hosted monorepo platform on plain Git. Projects come from a generator instead of copy-paste, coding agents work in isolated workspaces with guardrails enforced at the server, and every change — human or agent — lands through review and checks nobody can skip.

Request an invite

This instance is invite-only — ask and the code arrives by email. Or poke around first: the playground runs on sample data in your browser, and this instance's own source is open to read.

$ git clone … && docker compose up
  # your own control plane in ~2 minutes
